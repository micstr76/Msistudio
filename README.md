# Msistudio# Msistudio (BETA) 🚀
### Der visuelle Editor für das WiX Toolset v4

**Msistudio** ist ein modernes Tool für Windows-Administratoren und Software-Paketierer, die MSI-Installer effizient erstellen möchten, ohne sich im komplexen XML-Code des WiX Toolsets zu verlieren.

---

## 💡 Warum Msistudio?
Als IT-Systemadministratoren verbringen wir oft zu viel Zeit mit manuellem XML-Gefummel. Msistudio bringt die Mächtigkeit von **WiX v4** in eine moderne, visuelle Oberfläche. 

* **Kein XML-Zwang:** Erstelle komplexe Installer über eine intuitive GUI.
* **Deployment-Ready:** Fokus auf saubere MSI-Pakete für **SCCM & Intune**.
* **Fehlervermeidung:** Automatische Validierung und visuelles Feedback reduzieren Tippfehler in der Paketierung.

## ✨ Kernfunktionen
* **Visual Editor:** Intuitive Verwaltung von Dateien, Verzeichnissen und Registry-Keys.
* **EXE-to-MSI Wrapper:** Verpackt bestehende Setups (Inno Setup, NSIS, etc.) schnell in eine MSI-Struktur.
* **WiX v4 Engine:** Nutzt unter der Haube den neuesten Standard der WiX-Technologie.
* **Modernes UI:** Dunkles Farbschema (Dark Mode), optimiert für lange Arbeitstage.
